HTML List

an Unordered list

An Unordered list start with the <ul>tag.

Each list item define with <li> tag.

The list items will be marked with bullets (small black circles) by default:

example

  • cricket
  • football
  • hockey

an Ordered list

An ordered list starts with <ul>.tag.

The list items will be marked with numbers by default:

example

  1. cricket
  2. football
  3. hockey
